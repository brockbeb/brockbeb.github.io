Weather for Agriculture

python
flask
html
css

This was my final project for CS50. The agriculture industry is shifting more towards big farming. Fewer farmers remain, but the ones that do have larger acres to manage. With this shift in the industry, it can be difficult to get a specific forecast for a particular field in your operation. While most forecasts are more general given a zip code or city, this allows you to get a more accurate forecast for a particular field or location with latitude and longitude coordinates.

My goals for learning were to implement my first solo web app. I also wanted more experience in working with an API and JSON format. Beings I'm from the midwest, I love the weather. So, naturally, this was a great project for me to choose and something my uncle, who is a farmer, could utilize.
